# Security Policy

## Scope

This repository contains educational lab materials for a university course on cloud architecture.

The code is intentionally simplified for learning purposes. Some labs use configurations that would
not be appropriate for production environments.

## Reporting a Vulnerability

If you find a real security vulnerability in any script, configuration, or application code in this
repository, please **do not open a public GitHub Issue**.

Instead, report it privately via email: <alejandrogarcia@iteso.mx>

Include:

- The affected file or module
- A description of the vulnerability
- Steps to reproduce (if applicable)

## Response

Reports will be reviewed and addressed as course materials are updated. This is an educational
repository, so the priority will be to correct the material and add a learning note where relevant.
