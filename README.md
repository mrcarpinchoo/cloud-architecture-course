## 🚀 Cloud Architecture Course - ITESO

Welcome to the Cloud Architecture course repository! This repository contains all the materials, labs, and resources for the Cloud Architecture course taught at ITESO (Western Institute of Technology and Higher Education) during the Fall 2025 semester.

### 📚 Course Overview

This course focuses on applying programming, database, server management, and other knowledge within cloud environments. Students will learn about the cloud computing paradigm, its characteristics and benefits, understand the underlying complexity of configuring a private cloud, and learn to manage public clouds to optimize resource usage and reduce business costs.

### 🎯 Learning Objectives

- Understand the "Everything as a Service" concept
- Comprehend cloud service architecture
- Apply public cloud services in practical cases
- Design complex solutions using cloud services

The course is organized into weekly modules covering the following topics:

### 🧪 Labs and Workshops

The course includes hands-on labs to reinforce theoretical concepts:

1. Introduction to AWS IAM
2. Building VPCs and Launching Web Servers
3. Introduction to Amazon EC2
4. Working with EBS
5. Building Database Servers
6. Scaling and Load Balancing Architectures

|-----------|--------|

### 🛠️ Final Project

The final project requires students to work in teams of three to solve a real-world problem using cloud services. The project will be developed over 12 weeks, starting in week 4 and culminating in a presentation during week 17.

#### Project Requirements:
- Apply multiple concepts learned during the course
- Develop an original solution (may include third-party libraries and cloud services)
- Demonstrate sufficient complexity in code and architecture
- Include proper documentation and testing
