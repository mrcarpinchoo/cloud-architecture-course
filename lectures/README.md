# Lectures

This directory contains all lecture materials for the Cloud Architecture course.

## Structure

Each lecture is organized in its own folder with the following naming convention:

```
lecture-XX-topic-name/
```

Where XX is the lecture number (01, 02, etc.) and topic-name is a brief description of the lecture content.

## Contents

Each lecture folder typically contains:

- Slide deck (PDF and source format)
- Code examples (if applicable)
- Additional reading materials
- References to external resources

## Lecture Schedule

| # | Topic | Description |
|---|-------|-------------|
| 01 | Introduction to Cloud Computing | Overview of cloud computing concepts, service models, and deployment models |
| 02 | Cloud Economics and Billing | Understanding cloud pricing models, TCO, and cost optimization strategies |
| 03 | AWS Global Infrastructure | Exploring AWS regions, availability zones, and edge locations |
| 04 | Cloud Security | Cloud security principles, shared responsibility model, and security services |
| 05 | Networking and Content Delivery | VPCs, subnets, routing, and content delivery networks |
| 06 | Compute Services Part 1 | EC2, Auto Scaling, and related services |
| 07 | Compute Services Part 2 | Containers, serverless, and managed services |
| 08 | Storage Services | Block, object, and file storage options |
| 09 | Database Services | Relational, NoSQL, and specialized database services |
| 10 | Cloud Architecture | Well-Architected Framework and design principles |
| 11 | Auto Scaling and Monitoring | Scaling strategies, monitoring, and observability |
| 12 | Special Topic | Topic chosen by the class |

## Usage Guidelines

- Lecture materials are provided for educational purposes only
- Students are encouraged to take their own notes during lectures
- Code examples should be run in your own AWS account (free tier when possible)
- Additional resources are provided for deeper exploration of topics
